# ReactJS Test Application

## Что делает приложение
Приложение позволяет редактировать экземпляры двух разных моделей: Department(name) и Employee(firstName, lastName, departmentId)

## Использованный стек технологий:

* ReactJS
* React-redux
* React-router
* Gulp
* Webpack
* Babel
* Bootstrap
* SASS
* json-server
